# Campus Skill Match System
A web-based platform that connects university students with projects based on their skills.

## Tech Stack
- HTML/CSS - Frontend
- PHP - Backend
- MySQL - Database
- XAMPP - Local Server
- 
## Features
- Student register and login
- Add skills with levels (Beginner / Intermediate / Expert)
- Browse projects
- Get matched to projects based on your skills
- Apply to projects
- Track application status
- Admin panel to post projects and manage data
